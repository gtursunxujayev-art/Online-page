# Najot Nur Notiqlik Markazi - Frontend

React + TypeScript + Vite landing page frontend.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm run preview
```

## Structure

- `client/` - frontend source code
- `dist/` - production build output

## Notes

- This repository is frontend-only.
- API calls use `VITE_API_URL` when set, otherwise relative `/api/*` paths.
